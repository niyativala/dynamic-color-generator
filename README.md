# 🎨 Dynamic Color Generator

A simple web application that generates random RGB colors and applies them dynamically to the UI.  
This project demonstrates DOM manipulation, event handling, and dynamic styling using JavaScript.

---

## 🚀 Features

- 🎯 Generate random RGB colors on button click  
- 🌈 Displays the generated color code in real-time  
- 🎨 Applies color dynamically to the preview box  
- 🖱️ Interactive button with hover effects  
- 📱 Clean and centered user interface  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript 

---

## ⚙️ How It Works

1. Click the **Generate Color** button  
2. A random RGB color is generated using JavaScript  
3. The color code is displayed in the heading  
4. The preview box background updates instantly  

---

## 📂 Project Structure
index.html → UI structure  
style.css → Design and layout  
app.js → Functionality  

---

🎯 How to Use
- Clone the repository  
- Open index.html in your browser  
- Click the button to generate colors
